CSS (Cascading Style Sheets)

Css is used for designing the web pages. Css can be linked as a seperate file. This approach is called "External" or it can be added inside the style tag in HTML page as "Internal" and finally it can be added with style attribute in HTML elements. It is "Inline" way.
Here are some Css examples:
          
            h1, 
            h2 {
              font-weight: bold;
              font-family: 'Alegreya Sans';
              font-weight: 700;
            }
            h1 {
              font-size: 2em;
              font-family: 'Alegreya Sans';
            }
            h2 {
              font-size: 1.5em;
              font-family: 'Alegreya Sans';
            }
            
            p {
              margin-bottom: 1em; 
              font-family: 'Alegreya Sans';
              font-weight: 500;
            }
            article {
              font-size: 1em;
              font-family: 'Alegreya Sans';
              font-weight: 500;
            }
            
            footer {
              background-color: skyblue; 
            }
            .posts figure {
              overflow: hidden;
              height: 12em;
            }
            .blog-img {
              display: block;
              width: 100%;
              height: auto;
              border-radius: 50px 50px;
            }
            
            .logo_radius {
              border-radius: 50%;
            }
