HTML (Hypertext Markup Language)

HTML is a markup language. For web designers, three languages are essential which are HTML, CSS and finaly JavaScript. HTML contains the content of our web page. CSS allows us to design this page and JavaScript defines how it behaves.
HTML contains tags that each one has a unique task. For starting an HTML page, we need to type the basic tags like html, head and body. It is essential that we use "Semantic Tags". Here is the example of semantic tags inside the body element:
