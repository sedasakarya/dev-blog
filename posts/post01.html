HTML (Hypertext Markup Language)

HTML is a markup language. For web designers, three languages are essential which are HTML, CSS and finaly JavaScript. HTML contains the content of our web page. CSS allows us to design this page and JavaScript defines how it behaves.
HTML contains tags which each one has a unique task. If we open a tag, it is important to close it instantly to avoid forgetting to close it. For starting an HTML page,
